# C programming Project
---
## RUNNING ON WINDOWS: 
: Using gcc compiler:
Run the following commands in the terminal in order.
```
mkdir out
gcc ./src/*.c -o ./out/game.exe
copy src\words.txt out\
del *.obj
cd out
start ./game.exe
cd ..;;
```
